# Smart ToDo App

A full-stack todo application with AI-powered categorization and productivity tracking.

## Tech Stack

**Frontend:**
- React + Vite
- Tailwind CSS
- Axios for API calls
- React Router

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- bcrypt for password hashing

## Development Setup

### Prerequisites
- Node.js (v16 or higher)
- MongoDB (local or Atlas)
